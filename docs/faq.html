Find answers to common questions about Blizzity

General

Blizzity is a powerful visual automation platform specifically designed for video, audio, and image workflows. It allows you to create complex media processing pipelines without writing code.
Blizzity offers a free tier with 100 monthly credits for basic usage. For more advanced features and higher usage limits, we offer Pro ($10/month) and Studio ($48/seat/month) plans.
Blizzity is available for Windows (10/11), macOS (11 or later with Intel & Apple Silicon), and Linux (Ubuntu, Debian, Fedora, Arch). We also have a web app coming soon.
Blizzity is partially open source, with plans to open source more components in the future. Check our GitHub repository for the latest open source contributions.

Installation & Setup

You can download Blizzity from our website. Choose your platform (Windows, macOS, or Linux) and follow the installation instructions. We offer installers, portable versions, and package manager options.
Windows: Windows 10/11, 4GB RAM, 500MB disk space. macOS: macOS 11+, 4GB RAM, 500MB disk space. Linux: Modern distribution, 4GB RAM, 500MB disk space. 8GB+ RAM recommended for AI features.
Most features work offline, but AI nodes (ChatGPT, Stable Diffusion, etc.) require an internet connection as they connect to cloud services.
Blizzity checks for updates automatically. You'll be notified when a new version is available. You can also manually check for updates in Settings → About.

Credits & Billing

Credits are used to run workflows that utilize AI features and cloud processing. Different nodes consume different amounts of credits based on their computational cost. The Free plan includes 100 monthly credits.
You can purchase credit bundles directly from your account dashboard. We offer packages from $8 (500 credits) to $96 (5000 credits) with volume discounts.
Purchased credits do not expire and will remain in your account. However, you must have an active subscription to use them for running workflows.
Credit purchases are non-refundable, but we offer a full refund for subscription plans within 14 days if you're not satisfied.

Workflows & Nodes

Nodes are the building blocks of Blizzity workflows. Each node performs a specific function like loading a video, applying a filter, or generating AI content. You connect nodes to create automated pipelines.
Yes! You can share workflows by making them public in your project settings. Other users can then view, fork, and use your workflows.
AI nodes like ChatGPT, Stable Diffusion, and ElevenLabs require you to configure API keys in Settings. Some AI features are included with your subscription and don't require separate API keys.
Custom node development is coming soon! Currently, you can request new nodes in our community Discord or GitHub discussions.

Social Media Integration

Blizzity currently supports direct uploads to YouTube, TikTok, and Snapchat. More platforms are coming soon.
Go to Settings → Social Accounts and click 'Connect' for your desired platform. You'll be redirected to authenticate with the platform, then your account will be linked to Blizzity.
Yes! You can connect multiple accounts for each platform and switch between them when uploading content.
We use OAuth 2.0 for social media authentication and never store your passwords. Your access tokens are encrypted and stored securely.

Troubleshooting

Try running as administrator (Windows) or checking permissions (macOS/Linux). Make sure your system meets minimum requirements. Check our troubleshooting guide or contact support if the issue persists.
Check the error logs in the Run History panel. Common issues include insufficient credits, missing API keys for AI nodes, or invalid file paths. Make sure all required inputs are connected.
Check your export settings in the Render node. Make sure you're using appropriate bitrate and resolution settings. Higher quality exports may require more processing time.
Close unnecessary background applications, check that you meet system requirements, and ensure you have sufficient disk space. Complex workflows with AI nodes may require more processing time.
